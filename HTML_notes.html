Forms can be sent using "get" or "post"

Other HTML Markup

Block vs. Inline Elements

Block Elements: start new line

  • h1-6
  • p
  • ul
  • li

Inline Elements: continue flow on same line

  • b
  • em
  • a
  • img

iframe

A window carved out of your page

META tags

Types of meta tags, description; see head section for use

  • description: helps search engines know basic info about page, limit 155 characters
  • keywords: contains list of keywords separated by comma. used to impact how search engine indexed your site, no longer the case though
  • robots: indicates if search engines should add site to their results
  • author: names author
  • pragma: prevents browser from caching the page
  • expires: sets time for when page should expire and no longer be cached

Notes:

Positioning

Centering

  • To center boxes on page on within parent element, set left-margin and right-margin to auto
  • Because of the variety in screen sizes, web designers try to keep pgs between 960-1000px wide
  • Also, web designers try to fit the most important info in the top 570-600px vertically
